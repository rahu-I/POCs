# Proof of Concepts (POCs)

A collection of educational proof-of-concept projects demonstrating various web development concepts. It serves as a learning resource and reference. Each POC is self-contained with its own documentation.

## Projects Overview

### 1. [Payment Gateway](./payment-gateway/)

**iframe-based payment gateway architecture demonstrating web security concepts**

A comprehensive demonstration of how real-world payment gateways (like Stripe, PayPal, Razorpay) use iframe isolation to securely handle sensitive payment information for PCI DSS compliance.

[→ View detailed documentation](./payment-gateway/README.md)

---

### 2. [Create Vanilla App](./create-vanilla-app/)

**A lightweight CLI tool for scaffolding vanilla web applications**

A Node.js-based command-line tool that generates minimal web application structures without framework overhead. Created to learn how to build CLI tools with JavaScript.

[→ View detailed documentation](./create-vanilla-app/README.md)

---

### 3. [No Scaffold](./no-scaffold/)

**Demonstrating that `package.json` doesn't require complex build tools**

A minimalist project showing that `package.json` is simply a manifest file that npm reads - it can be used with even the simplest HTML projects for convenient script shortcuts.

[→ View detailed documentation](./no-scaffold/README.md)

---

## Getting Started

Each project is independent and has its own setup instructions. Navigate to the specific project directory and follow its README for installation and usage.
